useCallback 用来记忆一个函数。其实就是返回一个函数的 useMemo: `useCallback(realFunc, dependency)` 就等于 `useMemo(() => realFunc, dependency)` 
因为 useMemo 的固定写法是 `useMemo(function, dependency)`
第一个参数始终是一个 function，所以要返回一个函数的 useMemo 写法很难看 `useMemo(() => () => { console.log('要返回的函数') }, [])`
